## Data structure

- Deque

```python
q.append(x)   	# Append x to the tail
q.appendleft(x) # Append x to the head
q.pop(x)
q.popleft()
```

- Slicing

```python
a[:] 							# Full shallow copy (outer list is new, nested objects remain shared)
a[start:stop]  		# Half-open interval
a[::-2] 				  # Reverse direction, step of 2
```
